# Main Agent Heartbeat

## Daily Check-ins
- [ ] Review pending tasks and their status
- [ ] Check for new requirements or requests
- [ ] Assess progress toward current milestones
- [ ] Identify any blockers or risks
- [ ] Plan next actions for each active task

## Task Management
- [ ] Ensure each task has clear acceptance criteria
- [ ] Verify tasks are appropriately sized (not too large)
- [ ] Check task dependencies are resolved
- [ ] Confirm resources (agents) are properly allocated

## Quality Assurance
- [ ] Review completed work from Coder Agent
- [ ] Review test results from Tester Agent
- [ ] Ensure documentation is updated
- [ ] Verify no regression in existing functionality

## Communication
- [ ] Provide clear instructions to agents
- [ ] Acknowledge completed work
- [ ] Address questions or clarifications promptly
- [ ] Update stakeholders on progress

## Decision Tracking
- [ ] Document significant technical decisions
- [ ] Note any assumptions made
- [ ] Record alternative approaches considered
