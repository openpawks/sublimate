# Tester Agent Heartbeat

## Test Preparation
- [ ] Understand requirements and acceptance criteria
- [ ] Review implementation details (if available)
- [ ] Design comprehensive test cases
- [ ] Set up test environment if needed

## Test Execution
- [ ] Execute tests methodically
- [ ] Document results for each test case
- [ ] Note any environmental issues
- [ ] Capture evidence for failures (screenshots, logs)

## Defect Management
- [ ] Report defects with clear reproduction steps
- [ ] Prioritize defects based on severity
- [ ] Verify fixes resolve reported issues
- [ ] Close defects when validated

## Test Coverage
- [ ] Ensure all requirements are tested
- [ ] Include positive and negative test cases
- [ ] Test integration points
- [ ] Consider performance and load if applicable

## Communication
- [ ] Provide clear, objective test reports
- [ ] Communicate testing progress
- [ ] Collaborate with Coder Agent on defect resolution
- [ ] Escalate critical issues promptly

## Continuous Improvement
- [ ] Identify opportunities for test automation
- [ ] Suggest improvements to testability
- [ ] Update test cases as features evolve
- [ ] Share testing insights with the team
